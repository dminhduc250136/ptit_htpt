### GAP-01 — Documentation drift (severity: LOW)
**Vấn đề:** `.planning/REQUIREMENTS.md` còn show VAL-01/02/03 + FE-01 là `[ ]` unchecked dù Phase 03/04 đã PASS. Trace table ở cuối file consistent, chỉ checkbox lag. FE-01 status field còn ghi `needs-rework`.

**Hành động:** Flip 4 checkboxes + đổi FE-01 status từ `needs-rework` → `met`. (Sẽ được tự động xử lý trong bước archive requirements của `complete-milestone`.)

### GAP-02 — Phase 01 human verification pending (severity: LOW)
**Vấn đề:** `01-VERIFICATION.md` status=`human_needed` — Gates A-D (Swagger curl + UI smoke + status-code doc check) chưa được human tick thủ công.

**Mitigation:** Phase 04 UAT round-trips A3/A5 đi qua gateway thành công đã chứng minh ngầm: (a) gateway routes hoạt động, (b) envelope schema đúng, (c) /v3/api-docs codegen produce 6 generated files compile clean (đó là Gate A reachability via codegen). Gate B (validation envelope) đã được Phase 03 6 tests + Phase 04 B1 real backend cover.

**Hành động:** Waive Phase 01 human verification citing Phase 04 transitive coverage — KHÔNG cần re-run Swagger UI smoke vì Phase 4 đã prove tất cả integration điểm cần thiết.
