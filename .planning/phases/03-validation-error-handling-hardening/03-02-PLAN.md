| gateway -> downstream services | Error payload crossing service boundary may be malformed or overly rewritten |
| auth/authz mapper -> client | Authorization state must be represented correctly (401 vs 403) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-05 | I | Gateway/service auth errors | mitigate | Enforce explicit `UNAUTHORIZED` and `FORBIDDEN` code mapping with stable envelope |
| T-03-06 | R | Cross-boundary tracing | mitigate | Preserve request trace id in both gateway and service-origin error responses |
</threat_model>
