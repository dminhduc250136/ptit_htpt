## Spacing Scale

| Token CSS | Value | Usage trong Phase 8 |
|-----------|-------|---------------------|

| var(--space-2) | 8px | Cell padding trong order items table, badge margin |

## Typography

Toàn bộ dùng var(--text-*) và var(--weight-*) từ globals.css. Executor KHÔNG hardcode giá trị px.

**2 weights duy nhất được phép dùng trong Phase 8.**

| Role | Token CSS | Computed | Weight Token | Line Height Token | Dùng ở đâu trong Phase 8 |
|------|-----------|----------|--------------|-------------------|--------------------------|
| Page heading (h1) | var(--text-headline-sm) | 24px | var(--weight-semibold) 600 | var(--leading-tight) 1.2 | "Đơn hàng #{code}" trên order detail, "Chi tiết sản phẩm" |
| Section title (h3/h4) | var(--text-title-md) | 16px | var(--weight-semibold) 600 | var(--leading-normal) 1.5 | "Sản phẩm", "Địa chỉ giao hàng", "Thanh toán" headings |
| Body / table content | var(--text-body-md) | 14px | var(--weight-regular) 400 | var(--leading-relaxed) 1.6 | Line items rows, shipping address text, payment method text |
| Label / badge / meta | var(--text-body-sm) | 12px | var(--weight-semibold) 600 | var(--leading-normal) 1.5 | Stock badge text, order date subtitle, quantity label, table header row |

**Chú ý quan trọng cho Vietnamese:** Dùng `var(--leading-relaxed)` (1.6) cho body text — đã được optimize cho dấu tiếng Việt (nguồn: comment trong globals.css).

**Weight removed:** `var(--weight-medium)` 500 — không dùng trong Phase 8. Tất cả roles trước đây dùng weight-medium nay chuyển sang `var(--weight-semibold)` 600.

## Color

Toàn bộ dùng CSS custom properties từ globals.css. Executor KHÔNG hardcode hex values.

| Role | Token CSS | Hex tương đương | Usage trong Phase 8 |
|------|-----------|-----------------|---------------------|
| Dominant surface (60%) | var(--surface) / var(--surface-container-low) | #f7f9fb / #f2f4f6 | Page background, order detail page wrapper |
| Secondary surface (30%) | var(--surface-container-lowest) | #ffffff | Cards (infoCard, order items container), order breakdown box |
| Accent primary (10%) | var(--primary-container) | #0056d2 | "Thêm vào giỏ hàng" button background, CTA "Đặt hàng", active step indicator trong order tracker |
| In-stock positive | var(--secondary-container) | #fe6b00 | Badge/text "Còn hàng" — chỉ dùng cho stock positive indicator |
| Out-of-stock / error | var(--error) | #ba1a1a | Badge/text "Hết hàng", stock-shortage modal, disabled state message |
| Destructive | var(--error) | #ba1a1a | "Xóa khỏi giỏ" action trong stock conflict modal |
| Muted text | var(--on-surface-variant) | #424654 | Order date subtitle, meta labels, placeholder text |
| Total price emphasis | var(--primary-container) | #0056d2 | Giá tổng cộng (totalPrice) trong order breakdown — highlight duy nhất |

**Accent reserved for (danh sách đầy đủ — không dùng accent cho element khác):**
1. Button "Thêm vào giỏ hàng" (enabled state background)
2. Button "Đặt hàng" trong checkout (primary CTA)
3. Step indicator active dot trong order status tracker
4. Total price text trong order detail breakdown

**Màu semantic bổ sung:**
- `var(--secondary-container)` (#fe6b00 orange) dành riêng cho: badge "Còn hàng", rating stars (đã có trong product detail)
- `var(--error)` (#ba1a1a red) dành riêng cho: "Hết hàng" badge/text, error messages, destructive modal actions

| Trạng thái | Copy | Token màu | Vị trí |
|-----------|------|-----------|--------|
| `stock > 0` | "✓ Còn hàng ({n} sản phẩm)" | var(--secondary-container) — orange | Ngay dưới price block, trước quantity selector |

## Interaction Contract

### 1. Quantity Selector (Product Detail)

```
Trạng thái ban đầu: quantity = 1
Min: 1 (− disabled khi quantity === 1)
Max: product.stock (+ disabled khi quantity >= product.stock)
Input display: center-aligned, không editable trực tiếp
Touch target: min 44px × 44px cho mỗi button +/−
Transition: none (instant update)
```

**Bug fix D-13:** Điều kiện `disabled` hiện tại trên button "Thêm vào giỏ hàng" và +/− cần debug. Sau fix: disabled chỉ khi `product.stock === 0`.

### 2. Order Items Table Layout (Order Detail)

```
Layout: table hoặc CSS grid 4 cột
Col 1 — Sản phẩm: flex-grow 1, text-align left
Col 2 — Số lượng: width 80px, text-align center
Col 3 — Đơn giá: width 120px, text-align right
Col 4 — Thành tiền: width 120px, text-align right, font-weight semibold
Row separator: 1px solid var(--outline-variant)
Header row: var(--text-label-md), var(--weight-semibold), var(--on-surface-variant)
Data rows: var(--text-body-md), var(--weight-regular), var(--on-surface)
Tổng hàng cuối (totalRow): var(--text-title-md), var(--weight-semibold), col 4 color: var(--primary-container)
```

### 3. Order Status Tracker

Giữ nguyên pattern hiện tại trong page.tsx (trackerStep, trackerDot, trackerLine). Không thay đổi structure. Wire với real data từ `getOrderById`.

### 4. Loading States

| Context | Pattern |
|---------|---------|
| Order detail page fetch | Full-page skeleton: header bar 200px, 3 item rows 40px each, 2 info cards 120px each |
| Product stock fetch | Inline skeleton 80px × 16px trong .stockInfo div |
| Add-to-cart button | Button `loading={true}` prop (pattern đã có trong Button component) |
| Checkout submit | Button `loading={loading}` đã có — giữ nguyên |
