**Header (line 1, exact):**
```
id,category,photographer,license_note
```

**Rules theo D-18, D-19, D-20:**
- `id`: Unsplash photo ID dạng `1505740420928-5e560c06d30e` (NOT full URL — chỉ phần sau `photo-` và trước query params).
- `category`: chính xác 1 trong 5 giá trị: `phone`, `laptop`, `mouse`, `keyboard`, `headphone` (lowercase, không dấu).
- `photographer`: tên photographer hiển thị trên Unsplash page (có thể "Unknown" nếu không tìm được).
- `license_note`: ghi chú license. Mặc định: `Unsplash License (free use, no attribution required)`.

**Distribution requirements:**
- Phone: ≥20 rows (smartphone, iPhone, Samsung Galaxy, etc.)
- Laptop: ≥20 rows (MacBook, Dell XPS, ThinkPad, etc.)
- Mouse: ≥20 rows (computer/gaming mouse)
- Keyboard: ≥20 rows (mechanical keyboard, keychron-style)
- Headphone: ≥20 rows (over-ear, in-ear, gaming headset)
- Tổng tối thiểu 100 rows + header → file ≥101 dòng.

**Curation strategy (per RESEARCH Finding 5):**
1. Search Unsplash collections cho mỗi category (curator-managed photos ổn định hơn random search).
2. Verify mỗi photo ID bằng cách construct URL `https://images.unsplash.com/photo-{id}?fm=webp&q=80&w=800` và `curl -I` → expect HTTP/2 200, content-type chứa `image/webp`.
3. KHÔNG copy URL từ Unsplash API response có chứa `ixid=`/`ixlib=` — chỉ giữ phần ID gốc (Pitfall 6).
4. Mỗi photo ID xuất hiện đúng 1 lần (no duplicate).
5. Photo ID format hợp lệ: regex `^[a-zA-Z0-9_-]+$`, length 11-50 ký tự.

**Verification command (chạy sau khi tạo file):**
```bash
# Đếm rows mỗi category (expect ≥20 mỗi category)
awk -F',' 'NR>1 {print $2}' .planning/phases/16-seed-catalog-realistic/IMAGES.csv | sort | uniq -c

# Đếm tổng rows (expect ≥100 không kể header)
awk 'NR>1' .planning/phases/16-seed-catalog-realistic/IMAGES.csv | wc -l

# Verify không duplicate ID
awk -F',' 'NR>1 {print $1}' .planning/phases/16-seed-catalog-realistic/IMAGES.csv | sort | uniq -d
# Expect: empty output

# Spot-check 5 URLs random load 200 OK (sample, không bắt buộc 100/100)
for id in $(awk -F',' 'NR>1 {print $1}' .planning/phases/16-seed-catalog-realistic/IMAGES.csv | shuf -n 5); do
  curl -s -o /dev/null -w "%{http_code} %{content_type}\n" "https://images.unsplash.com/photo-${id}?fm=webp&q=80&w=800"
done
# Expect mỗi line: "200 image/webp"
```
