    ```java
    import com.ptit.htpt.productservice.domain.ReviewEntity;
    import org.springframework.data.jpa.domain.Specification;

    public final class AdminReviewSpecifications {
      private AdminReviewSpecifications() {}

      public static Specification<ReviewEntity> withFilter(String filter) {
        return (root, query, cb) -> switch (filter == null ? "all" : filter) {
          case "visible" -> cb.and(cb.isNull(root.get("deletedAt")), cb.isFalse(root.get("hidden")));
          case "hidden"  -> cb.isTrue(root.get("hidden"));
          case "deleted" -> cb.isNotNull(root.get("deletedAt"));
          default        -> cb.conjunction(); // "all"
        };
      }
    }
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Database ↔ JPA | DDL phải idempotent + reversible-safe; partial UNIQUE phải enforce sau migration |
| JPA ↔ Service | Visibility filter phải ở repository layer (KHÔNG để service call site quên `deletedAt IS NULL`) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-01-01 | Tampering | V7 migration on existing data | mitigate | DROP CONSTRAINT IF EXISTS + CREATE UNIQUE INDEX IF NOT EXISTS — idempotent. Partial UNIQUE re-enforces uniqueness cho data hiện trạng (deleted_at = NULL toàn bộ row). |
| T-21-01-02 | Information Disclosure | computeStats trả deleted/hidden vào avg | mitigate | JPQL WHERE clause cứng: `r.deletedAt IS NULL AND r.hidden = false` — không cho phép caller skip filter (D-08). |
| T-21-01-03 | Information Disclosure | Public finder leak deleted/hidden | mitigate | Method name `findByProductIdAndDeletedAtIsNullAndHiddenFalse` — Spring Data tự generate WHERE, không thể bypass ở caller. |
| T-21-01-04 | Repudiation | Migration không idempotent → re-run failure | accept | Flyway tracking handles re-runs; IF EXISTS guards bao phủ cả case manual repair. Single-developer demo project — repudiation risk thấp. |
| T-21-01-05 | Denial of Service | Index không tồn tại → seq scan trên admin filter | mitigate | `idx_reviews_visibility (product_id, hidden, deleted_at)` cover 4-state filter; partial UNIQUE cũng index `(product_id, user_id) WHERE deleted_at IS NULL`. |
</threat_model>
