**token.ts — 3 thay đổi:**

1. Thêm constant `ROLE_COOKIE` sau `PRESENCE_COOKIE`:
```typescript
const ROLE_COOKIE = 'user_role';
```

2. Sửa `setTokens(access: string, refresh: string)` → chấp nhận refresh optional:
```typescript
export function setTokens(access: string, refresh?: string | null): void {
  if (typeof window === 'undefined') return;
  window.localStorage.setItem(ACCESS_KEY, access);
  // Chỉ lưu refreshToken nếu có giá trị thật — tránh localStorage "undefined"
  if (refresh) {
    window.localStorage.setItem(REFRESH_KEY, refresh);
  }
  document.cookie = `${PRESENCE_COOKIE}=1; Path=/; SameSite=Lax; Max-Age=2592000`;
}
```

3. Thêm `setUserRole` và `clearUserRole` sau `clearTokens`:
```typescript
// user_role cookie — non-httpOnly, đọc bởi middleware.ts để check ADMIN (D-08)
// Max-Age=2592000 = 30 ngày (đồng bộ với auth_present)
export function setUserRole(role: string): void {
  if (typeof window === 'undefined') return;
  document.cookie = `${ROLE_COOKIE}=${role}; Path=/; SameSite=Lax; Max-Age=2592000`;
}

export function clearUserRole(): void {
  if (typeof window === 'undefined') return;
  document.cookie = `${ROLE_COOKIE}=; Path=/; SameSite=Lax; Max-Age=0`;
}
```

4. Cập nhật `clearTokens()` để gọi `clearUserRole()`:
```typescript
export function clearTokens(): void {
  if (typeof window === 'undefined') return;
  window.localStorage.removeItem(ACCESS_KEY);
  window.localStorage.removeItem(REFRESH_KEY);
  document.cookie = `${PRESENCE_COOKIE}=; Path=/; SameSite=Lax; Max-Age=0`;
  clearUserRole();   // per D-05: xóa user_role đồng thời
}
```
CRITICAL: `clearUserRole()` phải được định nghĩa TRƯỚC khi `clearTokens()` gọi nó (hoặc dùng forward reference nếu cần — nhưng đơn giản nhất là khai báo `clearUserRole` trước `clearTokens`).
