Nội dung 6 routes cần thêm:

```yaml
        # user-service admin: /api/users/admin → /admin/users
        - id: user-service-admin-base
          uri: http://user-service:8080
          predicates:
            - Path=/api/users/admin
          filters:
            - RewritePath=/api/users/admin, /admin/users
        - id: user-service-admin
          uri: http://user-service:8080
          predicates:
            - Path=/api/users/admin/**
          filters:
            - RewritePath=/api/users/admin/(?<seg>.*), /admin/users/${seg}
```

```yaml
        # product-service admin: /api/products/admin → /admin/products
        - id: product-service-admin-base
          uri: http://product-service:8080
          predicates:
            - Path=/api/products/admin
          filters:
            - RewritePath=/api/products/admin, /admin/products
        - id: product-service-admin
          uri: http://product-service:8080
          predicates:
            - Path=/api/products/admin/**
          filters:
            - RewritePath=/api/products/admin/(?<seg>.*), /admin/products/${seg}
```

```yaml
        # order-service admin: /api/orders/admin → /admin/orders
        - id: order-service-admin-base
          uri: http://order-service:8080
          predicates:
            - Path=/api/orders/admin
          filters:
            - RewritePath=/api/orders/admin, /admin/orders
        - id: order-service-admin
          uri: http://order-service:8080
          predicates:
            - Path=/api/orders/admin/**
          filters:
            - RewritePath=/api/orders/admin/(?<seg>.*), /admin/orders/${seg}
```

Lý do thứ tự: Spring Cloud Gateway match first-wins. Nếu admin routes đặt SAU general `Path=/api/users/**`, URL `/api/users/admin` sẽ bị route sang user-service với path `/users/admin` → 404.

Lý do two-route: Route base `Path=/api/users/admin` (không có `/**`) chỉ match exact path không có trailing segment — cần thiết cho `GET /api/users/admin` (list all users). Route wildcard `Path=/api/users/admin/**` match các path có segment thêm (`/{id}`, `/{id}/state`, v.v.).

Capture group name `seg` (không phải `path`) — tránh conflict với $PATH environment variable trong Docker container (đã xác nhận từ STATE.md Phase 5 Plan 08 decision).

Tìm method `listProducts(int page, int size, String sort, boolean includeDeleted)`. Thêm parameter `String keyword` vào cuối signature. Thêm filter vào stream chain SAU filter `!deleted` và TRƯỚC sort:

```java
public Map<String, Object> listProducts(int page, int size, String sort,
                                        boolean includeDeleted, String keyword) {
  List<ProductEntity> all = productRepo.findAll().stream()
      .filter(p -> includeDeleted || !p.deleted())
      .filter(p -> keyword == null || keyword.isBlank() ||
          p.name().toLowerCase().contains(keyword.toLowerCase()))
      .sorted(productComparator(sort))
      .toList();
  return paginate(all.stream().map(productMapper::toResponse).toList(), page, size);
  // (hoặc paginate theo cách cũ nếu không có productMapper — giữ y chang structure hiện tại)
}
```
