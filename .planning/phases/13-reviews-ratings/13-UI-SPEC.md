### StarWidget — Visual Spec

```
[★] [★] [★] [★] [★]
  ↑ mỗi <button>:
  - width: 44px; height: 44px (touch target min)
  - font-size: 24px (★ ký tự Unicode U+2605)
```

**States:**

| State | Fill color | Stroke / Outline |
|-------|-----------|-----------------|

| Selected (n <= value) | `--secondary-container` (#fe6b00) | none |
| Hovered trên selected | `--secondary-container` (#fe6b00) — giữ nguyên | none |

## Accessibility Contract

- **StarWidget:** `role="radiogroup"` trên container div + `aria-label="Chọn số sao"`. Mỗi `<button>` có `aria-label="{n} sao"` + `aria-pressed={value === n}` (pressed = sao này được chọn). Keyboard: Tab di chuyển giữa các buttons, Enter/Space để select.
- **ReviewForm:** `<label htmlFor="content">` linked đúng với `<textarea id="content">`. Submit button: `aria-disabled="true"` khi đang submitting. Char counter: `aria-live="polite"` để screen reader đọc update.
- **ReviewList:** `<ul>` với `role="list"` (hoặc native `<ul>`) cho list container. Mỗi review item là `<li>`. List heading: `<h3>` (trong hierarchy PDP tab content).
- **Avatar placeholder:** `aria-hidden="true"` (decorative initial letter).
- **Read-only stars (ReviewItem):** SVG polygon với `aria-hidden="true"` + sibling `<span className="sr-only">{review.rating} sao</span>` — screen reader đọc số sao, không cần đọc SVG.
- **Eligibility hint links:** `<a href="/login?redirect=...">` với descriptive text "Đăng nhập" — không dùng `<button>` style as link.
- **Load-more button:** `aria-label="Xem thêm đánh giá"` khi loading: thêm `aria-busy="true"`.
- **Error messages:** `role="alert"` trên rating required error (xuất hiện sau submit). Validation errors liên kết `aria-describedby` với input.
- **Hidden number input (rhf):** `aria-hidden="true"` + `tabIndex={-1}` — user không interact trực tiếp.
